Développement logiciel

Technologies : Java, Eclipse, Windev, Unity, C

Java m'a permis d'apprendre les bases de la programmation orientée objet. C'est aussi avec lui que j'ai appris l'optimisation du code.

Eclipse est un IDE que j'ai utilisé pour coder en Java.

Windev est un outil de développement très utilisé dans de nombreuses entreprises dû fait de sa fiabilité, de sa maintenabilité, et de sa rapidité de développement.

Unity est un moteur de jeu multiplateforme utilisant le C#. J'ai pu l'utiliser pour créer des jeux 3D et apprendre en autonomie les bases du développement de jeux-vidéos.

C est un langage de programmation de bas niveau. C'est avec lui que j'ai appris les bases de la programmation et de la gestion de mémoire.

Développement web

Technologies : SpringBoot, Angular, Laravel, Symfony, .NET

SpringBoot est un framework Java. J'ai notamment pu l'utiliser pour créer des API REST.

Angular est un framework JavaScript et c'est avec lui que j'ai appris à utiliser les composants.

Laravel et Symfony sont des frameworks PHP pour le développement d'applications web, largement utilisé dans la plupart des sites web.

.NET est un framework de développement qui utilise C#. J'ai pu l'utiliser pour créer des applications web dynamiques avec le patern MVVM.
